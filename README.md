# React+TypeScript+TailwindCSS Vite Template

# Technologies used

* React 19
* TypeScript 4
* TailwindCSS 4 (Vite Plugin)
* Eslint and Prettier
* Vite

# Getting Started
## Installation

Clone this repository with ```git clone https://github.com/Xenonyy/react-ts-tailwind-vite-template/``` \
Then run ``` npm install && npm run dev```
